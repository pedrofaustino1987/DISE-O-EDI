Programa de Estudio

Contenido del Diplomado

Seis módulos diseñados para dominar todas las áreas clave de la administración de infraestructura IT moderna.

01
Fundamentos de Redes
Protocolos TCP/IP, subredes, VLANs, routing y switching. Bases sólidas para entender la comunicación en infraestructuras.
02
Administración de Servidores
Linux y Windows Server, configuración de servicios, gestión de usuarios, permisos, directorio activo y políticas de seguridad.
03
Cloud Computing
AWS, Azure y GCP. Despliegue de máquinas virtuales, almacenamiento en la nube, redes virtuales y arquitecturas escalables.
04
Seguridad Perimetral
Firewalls, IDS/IPS, VPNs, segmentación de red, hardening de sistemas y buenas prácticas de seguridad defensiva.
05
Monitoreo y Observabilidad
Zabbix, Prometheus, Grafana, Nagios. Alertas, dashboards en tiempo real, métricas y logs centralizados con ELK Stack.
06
Infraestructura como Código
Terraform, Ansible, Docker y Kubernetes. Automatización de despliegues, configuración declarativa y CI/CD pipelines.
